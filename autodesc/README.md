# AutoDesc - Micro SaaS (Minimal Working Project)

Автоматический сервис генерации SEO-описаний для товаров.

**Содержание:** минимальный frontend, serverless backend для генерации описаний (OpenAI),
Stripe Checkout для оплаты, webhook для выдачи доступа, шаблоны промптов и писем.

**Цель:** дать полностью настроенный шаблон, который можно развернуть на Vercel / Render / Heroku.

## Быстрый старт (локально / Vercel)
1. Скопируйте репозиторий в свой компьютер или загрузите zip (в этом архиве).
2. Создайте аккаунт OpenAI и получите API-ключ.
3. Создайте аккаунт Stripe и получите секретный ключ и webhook secret.
4. (Опционально) Создайте базу данных Supabase/Postgres, получите URL и SERVICE_ROLE ключ.
5. Скопируйте `.env.example` в `.env` и заполните значения.
6. Установите зависимости:
   ```
   npm install
   ```
7. Запустите локально:
   ```
   npm run dev
   ```
8. Разверните на Vercel (или выбранной платформе) — добавьте переменные окружения.

## Что внутри
- `package.json` — зависимости и скрипты
- `api/` — serverless endpoints (generate, create-checkout-session, webhook)
- `public/index.html` — простая фронтенд-страница
- `prompt_templates.md` — готовые промпты для улучшения качества
- `email_templates.md` — шаблоны писем для продаж и поддержки

## Монетизация и ограничения
В этом примере показана модель покупки пакета через Stripe Checkout. Для production обязательно:
- Настроить HTTPS и проверку webhook-подписей Stripe.
- Настроить лимиты (rate-limits) и квоты для пользователей.
- Оптимизировать промпты и кеширование для снижения расходов на OpenAI.
